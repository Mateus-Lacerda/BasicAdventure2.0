# BasicAdventure
A game I've made to test some python concept.
How to run:
______________
WINDOWS:
________
Download all files and run the .exe file
_________________________________________________
Other platforms:
________________
Download all the ".py" files to a single folder
______
You must have python 3.10 or above
______
Open terminal
______
Find the folder in the cmd with the command "cd Folder/"
______
Run "python3 BasicAdventure.py"
